Guided student project groups through planning, collaboration, and implementation, providing technical support and coaching in the courses below:
* Systems Development (DIT113) 
* Team Programming (DIT094)

Key Responsibilities
======

* Supervised student teams working on software development and programming projects.
* Provided technical guidance on implementation, debugging, system design, and development practices.
* Supported project planning, task allocation, teamwork, and progress management.
* Reviewed student work and provided clear, constructive, and actionable feedback.
* Helped students apply software engineering theory in practical development environments.
* Encouraged independent problem-solving and collaborative learning.
* Assisted students in resolving both technical and team-related challenges.
* Communicated course expectations and clarified complex concepts during supervision sessions.


Skills Developed
======

Technical mentoring, software engineering, project supervision, team collaboration, problem-solving, communication, leadership, feedback delivery, conflict resolution, systems development, programming support, and academic instruction.
